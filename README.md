
## Installation

1. **Клонировать репозиторий:**
   ```bash
   git clone https://github.com/Artem-2RD22/JS_S4_KT.git
   cd JS_S4_KT
   ```

2. **Установить зависимости:**
   ```bash
   npm install
   ```

3. **Запустите сервер разработки:**
   ```bash
   npm run serve
   ```

## Структура проекта


- **src/**: исходные файлы.
 - **assets/**: изображения.
 - **components/**: компоненты Vue.
 - **store/**: конфигурация.
 - **views/**: страницы.
 - **App.vue**
 - **main.js**
